# Microsoft Foundry Demo

## Create Foundry

https://portal.azure.com

Create a New Resource | Microsoft Foundry | Fill in required fields.

This takes about 60 seconds, so you may want to create this in advance.

In the portal, show:

- "Projects" blade: 1 default project created
- "Access Control" blade: Role assignments
- "Diagnostics Settings"
- "Export template"
- "Overview" blade: [Go to Foundry portal]
